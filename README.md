
# 🎯 Student Performance Predictor

## 📌 Overview

This project uses Machine Learning to predict student exam scores based on study habits like study hours, sleep, and attendance.

## 🚀 Features

* Predict marks using real inputs
* Interactive command-line interface
* Data visualization (graph)

## 🧠 Machine Learning Model

* Linear Regression
* Train-test split for evaluation

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install pandas scikit-learn matplotlib
3. Run the project:
   python main.py

## 📊 Sample Output

* Model Accuracy displayed
* Predicted marks based on user input
* Graph showing study vs marks
